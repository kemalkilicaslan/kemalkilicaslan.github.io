Vehicle Speed Estimation

It is an object tracking and speed calculation application for estimating the speed of vehicles. First, the necessary libraries and modules are imported. Next, an image transformer class ViewTransformer is defined, which creates a perspective transformation matrix and transforms points using it. The source_video_path function is defined, which receives parameters such as the video path and thresholds. The main program retrieves the necessary video information using the video path and the corresponding threshold values. The YOLO model is used to detect objects within the video and these detections are used for the confidence threshold and for limiting objects within a given region. The object detection results are perspective transformed using an image converter. The velocities of the objects are calculated and labels of the objects tracked over several frames are generated. In each frame, a rendered image is created with tracking lines, bounding boxes and velocity labels of the objects.
