6174 Kaprekar Constant with Python Programming

6174, known as Kaprekar's Constant, is a mysterious number discovered by Dattatraya Ramchandra Kaprekar, a self-taught mathematician, in 1949. The number is reached through a process of repeatedly subtracting the smallest possible number from the largest possible number that can be formed using the digits of a given 4-digit number. This process continues until 6174 is reached, which then enters a repeating cycle where subsequent iterations result in the number 6174 again. The significance of 6174 lies in its mysterious properties, such as its ability to be reached from any 4-digit number and its tendency to trap subsequent calculations in a loop.
